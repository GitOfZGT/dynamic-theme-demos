# 动态主题模式 demo

vite + element-ui + scss


[在线演示](https://gitofzgt.github.io/dynamic-theme-demos/vite-element-ui-dynamic-theme/)

- Use with any UI framework(Based on less or sass),ant-design-vue just a example.
